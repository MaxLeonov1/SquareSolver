◆ DoubleCompare()

int DoubleCompare ( double double_1,
double double_2 )

Функция выполняет сравнение двух переменных типа double с учетом возможности накопления погрешности.

Функция принимает значения двух переменных типа double и сравнивает их разность с заданой в функции погрешностью. Функция возвращает значение int 1 если они равны, int 0 если нет.

Аргументы
[in]double_1первое число для сравнения
[in]double_2второе число для сравнения
Возвращает
1 если равны, 0 если нет

Функция для вывода сообщения о ошибки во время тестирования функций решения уравнения.

Принимает переменные типа double для значений коэффициентов уравнения , решнений и референсных решений, а также переменные типа int для количества решений и референсного количества решений. Выводит всю информацию в терминал. Функция ничего не возвращает

Аргументы
[in]coefficient_a,coefficient_b,coefficient_cкоэффициенты передаваемые функции решения
[in]solution_1,solution_2решения найденные функцией решения
[in]number_rootколичество корней найденное функцией решения
[in]ref_solution_1,ref_solution_2референсные решения
[in]ref_number_rootреференсное количество корней

◆ SetWorkingMode()

int SetWorkingMode ( void )

Функция задаёт режим работы программы (тестирование или решение)

Функция считывает из терминала символ. В случае символа Т возвращает int 0, в случае символа S - int 1, в иных случаях выводит сообщение о ошибке ввода и начинает процедуру выбора режима сначала.

Возвращает
1 если режим решения, 0 если режим тестирования
